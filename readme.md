A tool for detecting dependencies.

## Supports

- [x] Node.js
- [x] Bower
- [ ] Inline `<script>` tags
- [ ] CSS `@import`
- [ ] Ruby
- [ ] Python
- [ ] Go
- [ ] DNX
- [ ] ...

## Installation

```
$ npm install -g deedee
```

## Usage

```
$ deedee path/to/some/awesome/project
```
